How to use this file

1. Save as rules.axiom.yml in your project

2. Install: pip install axiomguard

3. Use in Python:

from axiomguard import KnowledgeBase, Claim

kb = KnowledgeBase()
kb.load_file("rules.axiom.yml")

claims = [Claim(subject="revenue", relation="value", object="500000")]
result = kb.verify(response_claims=claims)

print(result.is_hallucinating)  # True/False
print(result.reason)
