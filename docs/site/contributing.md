# Contributing

See the full [CONTRIBUTING.md](https://github.com/witchwasin/AxiomGuard/blob/main/CONTRIBUTING.md) on GitHub.

## Quick Start

```bash
git clone https://github.com/witchwasin/AxiomGuard.git
cd AxiomGuard
pip install -e ".[all,dev]"
pytest tests/
```

## Where to Contribute

- **NL-to-Logic translation** — entity normalization, complex sentences
- **New rule types** — comparisons, temporal logic, negation
- **New LLM backends** — Gemini, Mistral, Cohere
- **Benchmarks** — HaluEval, TruthfulQA
- **Examples** — new domain rule files and demo scripts

All ideas, feedback, and PRs are welcome.
